# 🧪 Agent Lab Skool — Website

The official website for [Agent Lab](https://www.skool.com/agent-lab), a community for builders and founders using AI to grow their businesses.

Built with Next.js, React, and Tailwind CSS.

## Getting Started

```bash
bun install
bun dev
```

## Stack

- **Next.js 16** with App Router
- **React 19** with Server Components
- **Tailwind CSS v4**
- **shadcn/ui** components
- **Motion** (Framer Motion) for animations
